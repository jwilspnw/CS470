LAB 5 - Mock Scheduler
----------------------
Usage:
Takes a file with PCB info, and schedules them down to completion, writing to a bin file at every iteration.
Takes a file called "processes.bin" and produces a file called "NEWproc.bin"

Recommended compilation:
gcc -Wall mockScheduler.c -o mockSched

TO GET MORE INFO FROM PROGRAM, SET "VERBOSE" AND/OR "DEBUG" TO 1 BEFORE COMPILATION
